# svn-scm

# Prerequisites

> **Note**: This extension leverages your 
> machine's SVN installation,  
> so you need to [install SVN](https://subversion.apache.org) first. 

## Features

- [x] Source Control View
- [x] Quick Diffs in gutter
- [ ] Status Bar
- [ ] SVN Commands
